# The server of scHLens


## how to start the project

Launching this project needs python==3.7.13 and R==4.2.2. The Environment variable R_HOME, which is the 安装目录 of R , needs to be set.

The library of Python and R also need to be installed:

```
pip install -r requirements.txt
```
and
```
Rscript package.R
```

At last, run this:
```
python app.py
```
